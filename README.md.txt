# Tournament results

----
##What's this project?

A project for [Udacity Full-Stack Web Developer Nanodegree](https://www.udacity.com/course/full-stack-web-developer-nanodegree--nd004), I'm enrolled in Novenber 2015.

Tournament results program is to store matche results in swiss pairing system.


## Operating Environment 
Python 2.7.10


## Files
* tournament.sql
* tournament.py
* tournament_test.py
* README.md


## How to start


**1. Download these files withe the command below**

----
    code git clone https://github.com/hananina/tournament.git

----

**2. Get to the download folder directory and run the command to run vagrant. and go to vm local file directory**

    vagrant up
    vagrant ssh
    cd /vagrant



**3. Run the command to run the SQL**

    psql 
    \i tournament.sql

**4. Get to the download folder directory and run the command to start the python programs**

----
    python tournament_test.py
